# Technologies frontend

## Framework :

La technologie frontend sera Angular

[site d'angular](https://angular.dev/)

### raison

il est un peu plus structuré que react, et utilise typescript par défaut, ce qui réduira les prises de tête.

## framework de style

daisyUI sera le framework de style

[daisyUI](https://daisyui.com/docs/install/)

### Raison

la syntaxe semble assez simple et je ne veux pas passer beaucoup de temps sur le styling.
