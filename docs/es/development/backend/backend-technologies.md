# Tecnologías del backend

## Lenguaje de programación:

Typescript

## Framework

El framework será [NestJs](https://nestjs.com/).
Nos permitirá crear una API para comunicarnos con el frontend.

## Integración de base de datos:

[Typeorm](https://typeorm.io/). 
Nos permitirá abstraer las consultas SQL y la conexión a la base de datos en funciones simples.
