# Technologies backend

## Langage de programmation :

Typescript

## Framework

Le framework sera [NestJs](https://nestjs.com/).
Il nous permettra de créer une API pour communiquer avec le frontend.

## Intégration de base de données :

[Typeorm](https://typeorm.io/).
Il nous permettra d'abstraire les requêtes SQL et la connexion à la base de données en fonctions simples.
